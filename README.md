# Zephyr Warehouse Management System

This repository contains the Zephyr Warehouse Management System, a Next.js application that provides comprehensive warehouse management functionality with a modern UI built on Shadcn UI and enhanced with custom template components.

## Project Structure

- **`/nextjs-app`** - The main Next.js application directory containing all source code

## Key Features

- Dashboard with real-time metrics and visualizations
- Inventory management with predictive analytics
- Order processing and tracking
- Warehouse efficiency monitoring
- Template-integrated UI components
- Responsive layout with mobile support
- Dark/light mode support

## Technical Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Shadcn UI components
- Chart.js for data visualization
- Supabase for backend services

## Deployment Information

This project is configured for deployment on Vercel with the root directory set to `/nextjs-app`.

## Development

For local development, navigate to the nextjs-app directory and run:

```bash
cd nextjs-app
npm install
npm run dev
```

The application will be available at http://localhost:3000
